# 關於我們
我們是以推動台灣語言學人才培育，促進語言跨領域研究與社會文化應用的公益性社會團體。

#### 我們的任務

1. 新世代語言學人才培育。
2. 獎助青年研究者從事語言學相關研究。
3. 提供語言相關之社會服務。
4. 促進台灣多元語言之發展。
5. 促進語言學跨領域之國內外交流活動。
6. 其他與章程所定宗旨及任務相關事項。
